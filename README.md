# rob137.github.io

Rob Kirby's blog. Jekyll on GitHub Pages, live at <https://rob137.github.io>.

Posts are compressed from ~25 minutes of dictation. Raw material lives in `_notes/`, ideas in `backlog.md`, voice and drafting workflow in `CLAUDE.md`.

New post: `_posts/YYYY-MM-DD-title.md`, date in `+0000`. A pre-push hook enforces GMT and non-future dates — after cloning, run `git config core.hooksPath .githooks`. Push to publish.
